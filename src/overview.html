This distribution contains the complete Java source code for the Elf online half-lead spliced composing applet, together with a simple application launcher not packaged the web version of Elf. Further documentation of the Elf source code and descriptions of the algorithms and optimisations implemented by this program can be found in the package and class javadoc.
